# Information Book App
An Android application developed using Java and Android Studio that displays informative content in an organized and interactive format. The app focuses on clean UI design, smooth navigation, and efficient content presentation.

Features: Categorized information display, RecyclerView integration, Image loading support, Interactive UI, Smooth navigation, Responsive design

Tech Stack: Java, Android Studio, XML, RecyclerView, Picasso/Glide (if used)

Purpose: This project was created to improve Android development skills including RecyclerView implementation, API/image handling, fragment management, and UI design.
